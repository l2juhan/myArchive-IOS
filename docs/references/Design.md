# 마이아카이브 (myArchive) — 디자인 명세 (Design Spec)

> 인디고 버전 기준 · 라이트 모드 전용 · iOS / SwiftUI 구현 대상
> 동봉된 `MyArchive_Design_Indigo.html`는 **의도된 룩앤필과 동작을 보여주는 디자인 레퍼런스**입니다. HTML을 그대로 이식하지 말고, 이 문서의 값으로 SwiftUI에서 **픽셀 단위로 재현**하세요.

본 디자인은 **고충실도(hi-fi)** 입니다. 색·타이포·간격·라운드·상태가 모두 확정값입니다.

### 1.4 라운드 / 간격 / 엘리베이션

- **Radius**: 카드 16 · 검색/Fill 12 · 리스트 아바타 12(42px) · 상세 아바타 17(62px) · 버튼 10~14 · 색 스와치 50%(30px) · 토글 31px pill(51×31, 노브 27) · 세그먼트 칩 7 · 모달 20 · 토스트 14 · 앱 아이콘 squircle(22.5%).
- **간격**: 헤더 상단 패딩 54(상태바 여백) / 좌우 20 · 카드 좌우 마진 14~16 · 행 패딩 11~14 · 필드 행 12~13 · gap 9~13 · 섹션 헤더 상 14~18 / 하 7.
- **엘리베이션**: 카드는 평면(그림자 없음). 주요 솔리드 버튼 `0 8px 20px rgba(70,71,174,.3)`. 토스트 `0 12px 32px rgba(0,0,0,.28)`. 모달 `0 24px 60px rgba(0,0,0,.32)`. 헤더는 `backdrop-filter: blur(20px) saturate(180%)` + 반투명 `rgba(247,248,251,.82)` + 하단 0.5px 헤어라인.

### 1.5 모션

| 이름 | 용도 | 값 |
| --- | --- | --- |
| maToast | 토스트 등장 | translateY(14→0)+opacity, .26s ease |

## 3. 인터랙션 & 상태 전이 (Behavior & State)

### 3.1 내비게이션 스택
- `stack: ['list']` 기반. list → detail → edit(상세에서 편집) / list → edit(신규) / list → settings. 잠금이 켜져 있으면 진입 시 `isLocked=true` 게이트.
- 뒤로가기는 스택 pop. detail을 벗어나면 `revealed` 초기화(재마스킹).

### 3.2 검색 (F-2)
- 입력 즉시 필터. 매칭 = `serviceName` 또는 `username` 부분 일치(대소문자 무시).
- 검색 중에는 즐겨찾기/전체 분리 없이 단일 “검색 결과” 목록. 클리어(✕)로 초기화.

### 3.3 정렬 (F-9 / F-14)
- 기본(`favoriteRecent`): 즐겨찾기 그룹을 최상단, 각 그룹 내부는 **최근 활동순**(= `updatedAt`, 없으면 `createdAt`).
- 이름순(`name`): 즐겨찾기 우선 없이 `serviceName` 오름차순(`localeCompare 'ko'`).
- 신규 저장 시 활동시각=현재 → 일반 그룹 최상단. 저장 직후 검색어는 초기화하여 새 항목이 바로 보이게 함.

### 3.4 타임스탬프 표시 — **‘사용’ → ‘수정/생성’으로 변경**
- 행/상세의 보조 텍스트는 **수정된 적이 있으면 “N분 전 수정”, 없으면 “N분 전 생성”**.
- 상대시간 규칙: <1분 ‘방금 전’, <60분 ‘N분 전’, <24시간 ‘N시간 전’, <30일 ‘N일 전’, 그 이상 ‘N주 전’.
- (기존 PRD의 `lastUsedAt` 기반 “최근 사용” 표기를 대체. 정렬의 ‘최근’ 기준도 `updatedAt`로 해석.)

### 3.5 마스킹 해제 (F-3)
- 시크릿 필드(아이디·비밀번호·커스텀)는 **기본 블러(blur 5px) + user-select 차단**. 눈 아이콘으로 탭 가능 힌트.
- **블러된 값 자체를 탭하면 그 필드만** 평문으로 표시(maReveal). 토글이 아니라 **단방향**.
- 약 22초 후 또는 상세 화면 이탈 시 **자동 재마스킹**. 메모/URL은 비시크릿(항상 표시).

### 3.6 필드별 복사 + 클립보드 만료 (F-4 / F-8)
- 필드마다 **독립 복사 버튼**(묶음 복사 없음). 탭 시 클립보드에 복사 + 토스트(“{라벨} 복사됨 · N초 후 클립보드에서 자동 삭제”) + 버튼 1.3s 체크.
- 만료 시간(설정값, 기본 60s) 경과 후 클립보드 자동 비움. 실제 iOS에서는 `UIPasteboard.expirationDate` 사용.

### 3.7 추가/수정 드래프트
- `draft`/`draftMode('new'|'edit')` 상태. 서비스명 비어있으면 저장 비활성.
- 커스텀 필드: 인라인 추가/삭제, 라벨·값 입력. 저장 시 라벨·값 모두 빈 항목은 폐기.
- 색상: 선택값 저장(미선택 시 서비스명 해시 폴백). 즐겨찾기 토글 포함.
- 저장 시 토스트(“저장됨”), 신규는 목록 최상단 노출.

### 3.8 삭제
- ‘계정 삭제’ → 확인 모달 → 확정 시 목록에서 제거 + 토스트. 되돌리기 없음.

### 3.9 잠금 / 인증 (F-5)
- `appLock` 기본 **꺼짐**. 설정 토글로 on/off. 켜면 진입 시 잠금 화면.
- “Face ID로 잠금 해제”/아이콘/“암호로…” → 인증 시뮬레이션(약 0.95s) 후 해제. “지금 잠그기”로 수동 잠금.
- 실제 구현: `LocalAuthentication`(LAContext), 백그라운드 복귀 시 재인증.

### 3.10 상태 변수 요약
`stack`, `isLocked`, `creds[]`, `query`, `sortMode`, `appLock`, `expiry(30|60|120)`, `detailId`, `revealed{fieldKey→bool}`, `draft`, `draftMode`, `pwShown`, `toast`, `copiedKey`, `confirmDelete`, `authing`.

## 4. 데이터 모델 (구현 매핑)

프로토타입은 상대시간을 위해 `createdAgo`/`updatedAgo`(분)로 표현 — 실제 구현에서는 `Date`로 대체.

**Credential**: `id`, `color`(HEX, 사용자 선택/폴백), `serviceName`, `username`, `password`, `url?`, `memo?`, `isFavorite`, `createdAt`, `updatedAt?`, `customFields[]`.
**CustomField**: `id`, `label`, `value`, `sortOrder`.

> 보안 저장(방식 B): 메타데이터는 SwiftData, 시크릿(비밀번호·커스텀 값)은 Keychain에 두고 메타에는 참조 키만. 자세한 내용은 PRD 7·9장 참조.
